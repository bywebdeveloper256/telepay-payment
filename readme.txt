== Description ==

Receive crypto payments with TelePay, create your merchant account at telepay.cash, get your API key and automate your invoices. TelePay has a web dashboard, Telegram integration and more so you can grow your business.

TelePay Woo plugin is the official plugin for telepay.cash.

This lugin integrates telepay.cash into your woocommerce store and adds a gateway so you can receive payments in TON, HIVE and HBD. We will add more cryptocurrencies soon.

In our telepay.cash Dashboard you can see the payments received and check your invoices.

The plugin seamlessly integrates with woocommerce so you can see statuses and orders in woocommerce dashboard as well.

TelePay Woo plugin automatically converts the amount from the store currency to the selected cryptocurrency to pay, and adds the note of the amount with the cryptocurrency in which the payment was made in the woocommerce order.

== Frequently Asked Questions ==

= Where do you find the Merchant API? =

Go to https://telepay.cash/, login or register and then go to the "Developers" tab and there you will see your secret key.

= Where do I see the payments received? =

Log in to telepay.cash and go to the "Invoice" tab. You can also create a new invoice from here.
